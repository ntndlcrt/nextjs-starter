## Getting Started

First, install the packages :

First, run the development server:

```bash
npm install

# or

yarn
```

The, run the server command :

```bash
npm run dev

#or

yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
